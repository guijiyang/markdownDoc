我给你做一张期货盘口成交术语终极对照表，只讲实战、准确、不绕，你看盘直接对照就行。
一、先记住 4 个基础动作

    多开（买开）：看涨，新开多单
    空开（卖开）：看跌，新开空单
    多平（卖平）：平掉多单
    空平（买平）：平掉空单

任何一笔成交 = 买方动作 + 卖方动作永远 1 买 = 1 卖，区别只在：谁和谁成交、谁主动、持仓怎么变。
二、终极成交对照表（直接背）
1. 增仓类（持仓 ↑）
① 双开 / 多开

    买方：多开
    卖方：空开
    结果：双方都是新开仓 → 增仓
    主动方是买方 → 软件标：多开
    本质：多头主动进攻增仓

② 空开

    买方：空平（老空离场）
    卖方：空开（新空进场）
    结果：净增仓
    主动方是卖方 → 软件标：空开
    本质：空头主动砸盘增仓

    一句话：多开、空开、双开，都是增仓，只看谁主动。

2. 减仓类（持仓 ↓）
① 多平

    买方：空平
    卖方：多平
    结果：多空都平仓 → 减仓
    软件标：多平
    含义：多头离场为主

② 空平

    买方：多开
    卖方：多平
    结果：减仓
    软件标：空平
    含义：空头平仓离场为主

3. 换手类（持仓不变）
① 多换

    买方：多开（新多来）
    卖方：多平（老多走）
    结果：多单换人，持仓不变
    标：多换

② 空换

    买方：空平（老空走）
    卖方：空开（新空来）
    结果：空单换人，持仓不变
    标：空换

三、实战一句话口诀

    增仓 = 有新资金进场
        多开：多头主动猛
        空开：空头主动猛
    减仓 = 资金离场
    换手 = 单子换人，没新增资金

四、你之前的疑问，最终定论

    多开、双开、空开都是增仓？多开是多头主动吃卖价，空开是空头主动吃买价？双开是什么？

最终标准答案：

    多开、空开、双开，都是增仓
    多开 = 多头主动吃卖价的双开
    空开 = 空头主动砸买价的增仓
    双开 = 多开 + 空开配对成交的统称

一、先记住 4 条核心公式（期货盘口真理）

    增仓上涨 = 多头真强（多开主导）
    增仓下跌 = 空头真强（空开主导）
    减仓上涨 = 空头在逃命（空平）
    减仓下跌 = 多头在割肉（多平）

二、逐一带你看懂盘口语言
1. 多开 —— 最强上涨信号

    现象：价格涨 + 持仓大增 + 主动多开多
    含义：
    新多头疯狂进场，新空头敢接，
    真金白银往里冲，上涨最稳、最猛。
    结论：强势多头，趋势大概率延续。

2. 空开 —— 最强下跌信号

    现象：价格跌 + 持仓大增 + 主动空开多
    含义：
    新空头疯狂砸盘，新多头敢接，
    资金一致看空，下跌最凶。
    结论：强势空头，跌势大概率延续。

3. 多平 —— 多头认输、弱势下跌

    现象：价格跌 + 持仓大减 + 多平多
    含义：
    之前做多的人扛不住，平仓跑路。
    结论：
    不是新空砸盘，是老多割肉，
    跌势可能快到位，但短期很弱。

4. 空平 —— 空头止损、被动上涨

    现象：价格涨 + 持仓大减 + 空平多
    含义：
    之前做空的人亏麻了，被迫买入平仓。
    结论：
    不是新多进攻，是空头止损，
    上涨容易反复，不算真强。

5. 多换 / 空换 —— 震荡、换人

    持仓不变，只是老的走、新的来。
    结论：方向不明，震荡为主。

三、最实用的一句话总结（背这个）

    增仓走方向，减仓走修复，换手走震荡。
    增仓涨 = 真涨
    增仓跌 = 真跌
    减仓涨 = 空头跑
    减仓跌 = 多头跑
